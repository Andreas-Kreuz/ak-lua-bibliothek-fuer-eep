# Demo-Anlagen

Dieses Verzeichnis enthält fertige Beispielanlagen, mit denen Du die Funktionen der Control Extension direkt ausprobieren kannst.

## Enthaltene Demos

- **ampel/** - Demo-Anlage für die automatische Ampelsteuerung mit `ce.mods.road`
- **demo-linien/** - Demo-Anlage für ÖPNV-Linienführung mit `ce.mods.transit`
- **testen/** - Testanlagen und Hilfsskripte für die Entwicklung
- **tutorial-ampel/** - Schritt-für-Schritt-Tutorial für eine einfache Ampelkreuzung

## Verwendung

1. Lade die gewünschte Anlage herunter
2. Kopiere die Lua-Dateien in das `LUA`-Verzeichnis von EEP
3. Öffne die Anlage in EEP und starte die Simulation
